# 게시판 이름 : "My ground"
게시판 기능 구현 담당

- 사용언어 - JAVA,HTML/CSS
- 프레임워크 - SpringBoot, JPA
- DB - MariaDB
- IDE - IntelliJ, Navicat workbench


# 구현 기능
- 백엔드: 글 목록 조회, 글 상세보기(모달 이용), 글 작성, 수정, 삭제, 페이지네이션, 검색 기능을 구현하였습니다. 
- 프론트엔드: 인스타그램의 웹 페이지 중 본인 프로필 페이지를 참고하여 만들어보았습니다.
